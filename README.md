# Japanese Kintsugi Restoration Registry

This registry documents traditional Japanese kintsugi restoration techniques.  
Kintsugi is the practice of repairing broken ceramics using urushi lacquer and metallic powders, often gold. It is both a technical craft and a cultural philosophy that treats breakage as part of an object's history.

## Purpose

The goal of this contract is to preserve essential kintsugi techniques in a permanent and verifiable format.  
These methods are transmitted through specialized lacquer workshops and restoration lineages, and many are vulnerable as artisan communities shrink.

By recording selected techniques on-chain, the registry protects cultural knowledge from loss or distortion and keeps the connection between technique, material, and symbolic meaning.

## Why this contract matters

Kintsugi is widely referenced but often misunderstood.  
Its real practice involves slow, precise urushi work, multiple curing stages, and deep aesthetic intention.

This registry helps by:

- preserving verified information in a neutral and permanent format  
- keeping cultural context attached to each technique  
- offering a reference for artisans, researchers, and cultural institutions  
- preventing the simplification or misrepresentation of traditional restoration methods  

## What makes this registry unique

The registry focuses on the restoration process itself, not on the finished repaired object.  
It treats bonding, filling, layering, and metallic finishing as cultural assets worth preserving.

The contract is intentionally minimal.  
It records only the essential information needed to identify and contextualize each technique.

## How it works

Each entry includes:

- region  
- workshop  
- technique name  
- material type  
- cultural note  
- ritual or symbolic use  

A simple like and dislike system allows basic community feedback.
